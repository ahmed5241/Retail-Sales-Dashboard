# Retail Sales Analysis Project

## Overview

This project demonstrates a complete data analysis workflow, starting from data exploration and analysis in Python to building an interactive Power BI dashboard for business decision-making.

## Project Workflow

### 1. Data Exploration & Cleaning

* Data quality assessment
* Data type validation
* Missing value analysis
* Feature engineering

### 2. Exploratory Data Analysis (EDA)

Performed using Python libraries:

* Pandas
* Matplotlib
* NumPy

Key analyses included:

* Product category performance
* Customer demographics
* Monthly sales trends
* Top customers analysis
* Age group segmentation

### 3. Business Insights

* Electronics and Clothing generated the highest revenue.
* Customers aged 46–55 contributed the most revenue.
* Sales performance varied across different months.
* A small group of customers generated a significant portion of total revenue.

### 4. Power BI Dashboard

The dashboard includes:

* KPI Cards
* Revenue Analysis
* Monthly Trends
* Customer Segmentation
* Top Customers Analysis
* Interactive Filters
* Dynamic Titles
